# zss-ca

Zenith Strategic Solutions agency website powered by Next.js 16, Supabase, and shadcn/ui.

## Repo & Deployment

- GitHub: [zssca/zss-ca](https://github.com/zssca/zss-ca)
- Local directory: `zss-ca`
- Vercel project: `zss-ca`

## Getting Started

```bash
npm install
npm run dev
```

Build and smoke test before deploy:

```bash
npm run build
npm run start
```

Set environment variables in `.env.local` (Supabase keys, Stripe keys, Resend, etc.) prior to running production builds or deploying to Vercel.
